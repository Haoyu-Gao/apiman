Policies

Rate Limiting Policy

You will be rate limited to {{rateLimitPolicy.limit}} requests per {{rateLimitPolicy.unit}}.

The following headers in the response contain useful information relating to rate limit

  • X-RateLimit-Limit: Rate limit allowance (per time period)
  • X-RateLimit-Remaining: Rate limit allowance remaining
  • X-RateLimit-Reset: Time until next rate limit reset
 Transfer Quota Policy

You will be rate limited to {{quotaLimitPolicy.limit}} requests per {{quotaLimitPolicy.unit}}.

The following headers in the response contain useful information relating to quota limit

  • X-TransferQuota-Limit: Quota allowance (per time period)
  • X-TransferQuota-Remaining: Quota remaining
  • X-TransferQuota-Reset: Time until quota reset
